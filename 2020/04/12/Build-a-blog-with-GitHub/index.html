Published on: Apr 12, 2020
Tags: github
一、前提条件#

1.拥有一个GitHub账号

2.安装了ruby环境（版本在2.4以上）

二、准备工作——安装Jekyll#

1.执行命令 ruby -v，查看ruby版本，是否为2.4.0或更高版本

2.执行 gem -v,查看RubyGems检查版本

3.执行命令，安装jekyll和bundler
